### 安装方式

    克隆本仓库后请自行配置python环境
    本项目需要以下python库:
        flask,flask_cors,json,
        time,threading,os,apscheduler,
        shutil,pathlib,os,pydub,sys
    同时，请在你的系统中安装ffmpeg


#### 服务端指令

| 指令           | 功能说明       |
|:-------------|:-----------|
| **ls**       | 列出所有房间信息   |
| **rm [房间名]** | 删除指定房间     |
| **exit**     | 退出服务端并关闭程序 |
